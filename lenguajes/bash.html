Bash

    ¿Cómo comenzar?

    Ejecuta el comando:
    nano ~/.bashrc
    Para ver todos los alias una vez dentro del archivo
    .bashrc
    vas a encontrar algo así:
    alias actualizar=sudo apt update && apt full-upgrade

    Comandos editados de BASH

    iniciar apache =
    apacheini
    cerrar apache =
    apachefin
    reiniciar apache =
    apacheres
    iniciar postgresql =
    postgresini
    finalizar postgresql =
    postgresfin
    sudo apt update && apt full-upgrade =
    actualizar
    history -c && history -w && rm ~/.bash_history =
    rmhistorial
    sudo sync && sudo sysctl vm.drop_caches=3 =
    liberar
    xdotool getmouselocation =
    mousefind
    sudo service mysql start =
    sqlini
    sudo service mysql stop =
    sqlfin

    Cambiar case sensitive de la terminal

    shopt -s nocaseglob
    Activar modo case-insensitive
    shopt -u nocaseglob =
    Volver al modo case-sensitive

    Copiar archivos de una carpeta a otra

    sudo cp
    /llenar/llenar/DondeEstá/archivo.txt /llenar/llenar/DondeLoLLevo

    Ver archivos modificados

    en los últimos 3 días
    find / -type f -mtime -3
    en los últimos 30min
    find / -type f -mmin -30

    Copiar archivos de una carpeta a otra

    Ver numeración de lineas en nano
    alt + ctrl + 3
    Abrir nueva terminal
    Ctrl + Alt + T
    Ocultar menu de terminal
    gsettings set org.gnome.Terminal.Legacy.Settings default-show-menubar false

    Cambiar MAC

    Dar de baja la MAC
    sudo ifconfig (NOMBRE DE LA NET) down
    Cambiar la MAC
    sudo macchanger -r (NOMBRE DE LA NET)
    Dar de alta la MAC
    sudo ifconfig (NOMBRE DE LA NET) up

    Codificaciones Base64 y Hex

    Códificar texto base 64
    echo "MI TEXTO" | base64
    Decodificar texto base 64
    base64 -d
    Códificar Hexadecimal ->
    echo "MI TEXTO" | xxd
    Decodificar Hexadecimal ->
    echo "TEXTO HEXA" | xxd -r

    Conectarse a dispositivos por ADB

    ¡Debes estar en la misma red WiFi!

    Buscar dispositivos
    adb devices
    Conectados por cable/usb
    adb -s CÓDIGO_DEL_DISPOSITIVO shell
    Ejecuto el script
    ./adbRecon.sh

    Comandos para ejecutar

    Abrir una página web
    adb shell am start -a android.intent.action.VIEW -d "PAGINA WEB"
    Realizar una llamada
    adb shell am start -a android.intent.action.CALL -d tel:NUMERO DE TEL
    Ver cámara Frontal
    adb shell am start -a android.media.action.IMAGE_CAPTURE
    Grabar audio
    adb shell am start -a android.provider.MediaStore.RECORD_SOUND
    Sonar alarma
    adb shell am broadcast -a
    Apagar dispositivo
    android.intent.action.ACTION_REQUEST_SHUTDOWN
    Reiniciar dispositivo
    adb shell reboot -p
    Botón de retroceso
    adb shell input keyevent KEYCODE_BACK
    Tomar captura del dispositivo
    adb shell screencap -p /sdcard/captura_de_pantalla.png
    Guardar captura en PC
    adb pull /sdcard/captura_de_pantalla.png /carpeta/donde/quieras/

    Identificar Sistema Operativo

    Buscar dispositivos
    ping -c 1 IP
    Windows
    ttl 118
    Linux
    ttl 60/70

    Netdiscover

    Netdiscover es una herramienta de escaneo de red que se utiliza para descubrir y mostrar dispositivos en una red local (con esta se puede optener la MAC)

    Obtener interfaz
    ifconfig
    Linux
    netdiscover -i Interfaz -r IpPrivada

    Virus total scanner

    Sirve para escanear archivos y detectar si es malicioso

    Escaneo
    vtscan ruta/del/archivo

    Host request

    Sirve para escanear una página web y obtener la ip

    Escaneo de la página web
    host www.mipaginaweb.com

    Arpscanner

    Sirve para obtener la MAC de una red mediante la ip (eth -> MAC)

    Busqueda de MAC
    Ip link show

    Activar/Desactivar SSH

    Detener SSH
    sudo systemctl stop ssh
    Iniciar SSH
    sudo systemctl start ssh
    Detener SSH
    sudo service ssh start
    Iniciar SSH
    sudo service ssh stop

    Desactivar ipv6

    Detener ipv6
    sudo sysctl -w net.ipv6.conf.all.disable_ipv6=1 && sysctl -w net.ipv6.conf.default.disable_ipv6=1

    Network Manager

    Var conexiones de red activas
    nmcli connection show --active
    Var redes WiFi
    nmcli device wifi list
    Var redes WiFi a la que estamos conectados
    nmcli d
